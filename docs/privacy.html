Privacy Policy

BGA Turn Bot · Last updated: 2026/07/14

This Privacy Policy explains what data BGA Turn Bot (the "Bot") stores, why, and how it is handled. The Bot is a self-hosted Discord bot that watches public Board Game Arena tables and posts turn-status messages in Discord. It is designed to store the minimum data needed to function.

1. Data we store

The Bot stores the following data in a local database on the server where it runs:

  • Discord identifiers: user IDs, server (guild) IDs, and channel IDs, used to know where to post messages and which member is linked to which player.
  • Board Game Arena player information: the public BGA player name and/or public BGA player ID that a server administrator manually links to a Discord member.
  • Watched table information: the public BGA table URLs and table IDs added via the /bga watch command, along with the last known public game state (such as whose turn it is and the game name).

The Bot does not collect message content, private messages, email addresses, passwords, payment information, IP addresses, or any Board Game Arena login credentials. It only reads publicly available BGA table pages in spectator mode, without cookies or a BGA account.

2. How the data is used

The stored data is used solely to operate the Bot's core features: linking a Discord member to a BGA player, watching public tables, detecting turns, and creating, updating, or deleting Discord status messages accordingly. The data is not used for advertising, profiling, or any purpose beyond running the Bot.

3. Where the data is stored

All data is stored locally in an SQLite database file on the server operated by the person self-hosting the Bot. It is not sold, rented, or shared with third parties, and it is not sent to any external analytics or advertising service.

4. Data retention and deletion

You can remove your data at any time using the Bot's commands:

  • /bga unlink-member removes the link between a Discord member and a BGA player.
  • /bga unwatch and /bga unwatch-all remove watched tables and their stored state.

When the Bot is removed from a server, the operator can delete the associated data. For a complete deletion request, contact the operator using the details below.

5. Third-party services

The Bot operates on Discord and reads public data from Board Game Arena. The Bot is not affiliated with either service. Your use of Discord is governed by Discord's Privacy Policy, and your use of Board Game Arena is governed by its own privacy terms.

6. Children

The Bot is intended to be used within Discord, which requires users to be at least 13 years old (or the minimum age of digital consent in their country). The Bot is not directed at children below that age.

7. Changes to this policy

This Privacy Policy may be updated from time to time. The "last updated" date at the top reflects the most recent revision. Continued use of the Bot after changes take effect constitutes acceptance of the revised policy.
